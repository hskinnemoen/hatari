Hatari links

Here are some hyperlinks related to Atari ST and 68000 emulation :

Hatari related sites:

  • Marco Herrn's Debian packages : Hatari .deb packages for the Debian "testing" and "unstable" distributions.
  • Hatari for N770. : Eero Tamminen has ported Hatari to the N770/Maemo.
  • BeEmulated : Emulators for BeOS, here you can find the binary version of Hatari for BeOS.
  • Hatari for AmigaOS : Ventzislav Tzvetkov has ported Hatari to AmigaOS.
  • Hatari for Acorn RISC OS : James Lampard has ported Hatari to the Acorn RISC machines.
  • Norman Feske's Site : Norman has ported Hatari to the DoPE/Drops system.

Other emulators:

  • STonX : A good (especially for running applications) Atari ST emulator for Linux/Unix.
  • Castaway (a.k.a. FAST) : Yet another Atari ST emulator for Unix.
  • Aranym : The "Atari running on any machine" project.
  • Osis : Osis is a combination of a ST emulator and a TOS library for Linux.
  • STeem : STeem is a very good STE emulator for Linux and Windows, it runs most of the old games and demos without problems. You can also find developer documentation about the ST on their web site.
  • WinSTon : WinSTon is an Atari emulator for Windoze, but its sourcecode is available. Hatari is based on this sourcecode!
  • UAE : The UN*X Amiga Emulator - Hatari uses its CPU kernel.
  • Experimental UAE : An unofficial, improved version of UAE.
  • NoSTalgia : NoSTalgia is a nice ST emulator for the Macintosh computers.
  • Generator : A Sega Genesis emulator. Might be interesting because it features a nice 68000 emulation, too.
  • Virtual Jaguar : An Atari Jaguar emulator for Linux.

ST emulation and other related sites:

  • Atari.org : One of the best Atari portals on the web. It features a nice discussion forum, too.
  • Atari Legend : A huge data base about Atari ST games.
  • PaciDemo : A page dedicated to demos on ST emulators. You can download lots of disk images with scene demos here.
  • Persistence Of Vision : POV (not the raytracer!) have created a lot of demo menu disks which can be downloaded from their pages.
  • EmuTOS : The people of this project try to rewrite a free TOS clone for emulators.
  • Pangaelin Willow TOS ROM page : Offers almost all TOS ROMs.
  • Atari forums : Very good Atari ST discussion forums.
  • sc68 : A music player for Atari ST and Amiga soundchip musics.
  • Dead Hacker Society : Excelent news site and demo scene BBS.
  • Sparemint : RPM based software distribution for FreeMiNT. You can also find a GCC cross compiler for the ST on this site.

Other usefull tools for the ST emulation:

  • Socat : This tool can emulate a serial device over a TCP/IP network connection - usefull for playing some ST games which support multi-player via serial line.
  • dd_rescue and GNU ddrescue : Tools for creating images from floppy disks.
  • Mtools : Tools for accessing a FAT filesystem - usefull for floppy disk images.
  • Vincent Rivière's m68k-atari-mint tools : GCC 4.2 tool chain for cross compiling software for the ST.

Documentation and information sites:

  • Atari Document Archive : Great resource of Atari developer and other documentation files.
  • The Atari ST Internals : Exception vectors, system variables, memory maps, ...
  • "Lovely" ST docs : Lots of documentation about the 68000 CPU and the ST hardware. You can also find an Atari ST emulator written in Java there. However, many documentation links on the page seem currently to be broken. (try www.archive.org to see some of the documentation files).
  • The Atari compendium : Huge documentation about the TOS.
  • MiKRO's website : Lots of good Atari documentation (also about the Falcon!) can be found here.
  • Sigma Soft : Here you can find the great "Turbo Assembler" and a (German) hypertext documentation about the BIOS and XBIOS of the Atari ST.
  • TOS Hypertext : The "TOS Hypertext" is a great (German) documentation about the GEM, GEMDOS, XBIOS, BIOS and more.
