Hatari screenshots

STE games and demos:

The Atari STE has some additional graphic and sound capabilities compared to the normal ST. Here are some screenshots from STE programs which Hatari can run...


The Mind-Rewind demo, a demo which heavily uses the additional features of the STE shifter. The demo works quite fine in Hatari apart from some few graphical glitches due to timing problems.


Screenshot from the Obsession demo. Obsession is a great flipper game for the STE which uses a STE shifter bug for displaying more graphics than usual in the right border.


Stardust tunnel demo.


The Pacemaker demo from Paradox - certainly one of the best STE demos.


The Braindamage demo.

Utopos - this game uses the STE splitscreen technique.
